#Hospitals

Es una app hecha con Delphi7 / Object Pascal donde su principal objetivo es la gestion de:

- hospitales
- obras sociales
- medicos
- auxiliares
- enfermeras
- habitaciones del hospitales
- internaciones
- cobros/pagos
